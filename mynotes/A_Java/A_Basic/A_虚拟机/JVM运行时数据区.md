# JVM运行时数据区

## 虚拟机栈(VM Stack)
* 虚拟机栈生命周期与线程相同，它描述Java方法执行的内存模型。
* 每个方法在执行的同时都会创建一个栈帧来存储局部变量表，操作数栈，动态连接和方法出口信息。
* 每个方法从调用到执行完毕的过程，对应一个栈帧在虚拟机栈中从入栈到出栈的过程。
