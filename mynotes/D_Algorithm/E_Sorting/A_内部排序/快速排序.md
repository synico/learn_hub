### 冒泡排序 (bubble sort)
#### 描述
#### 实现
```
public static <AnyType extends Comparable<? super AnyType>> void quickSort(AnyType [] a) {
  for(int i = 0; i < a.length; i++) {
    int temp;
    for(int j = 0; j < a.length - i - 1; j++) {
      if(a[j] > a[j + 1]) {
        temp = a[j];
        a[j] = a[j + 1];
        a[j + 1] = temp;
      }
    }
  }
}
```
